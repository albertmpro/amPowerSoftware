amDevFlow Notes 
---------------------------------
An IDE designed to devlop websites do basic art and browse the web when nessasay. The main interface is Tab basied, with my own custom MVVM back end. 

MVVM Notes 
----------------------
FlowViewModel is the main ViewModel used to link the diffrent task to the main application.  

TextViewModel is used for WriterFlow and CodeFlow to do basic functions such as save and load Text files as well as other method's to setup quick templates esay. 

InkViewModel is used to handle the Ink information that I want to save and reuse again, the export method when I want to create an Image 

BrowserVeiwModel is used to manage the backend for BrowserFlow Applications 

WebMsgViewModel is used to manage the save and load information for the WebMsg Image
